# Apple & Orange

For more information on this HackerRank challenge follow this [link](https://www.hackerrank.com/challenges/apple-and-orange/problem?isFullScreen=false)

Run this solution using:
```php
php solution.php
```

Following this provide the following example inputs to the console:
```bash
7 10 # Two integer values representing the house starting & ending points (Press enter)
4 12 # Two integer values representing the apple tree & orange tree locations respectively (Press enter)
2 3 -4 # n distances from the tree an apple has fallen
3 -2 -4 # n distances from the tree an orange has fallen
```

Expected output:
```bash
1
2
```
