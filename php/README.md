## Project Structure
- `algorithms` 
For each problem set the project structure is as follows:
- `problem_name/`
    - `solution.php` The PHP solution code 
    - `input/` The input .txt files
        - `input00.txt` 
    - `output/` The expected output .txt files
        - `output00.txt`

The project contains a global test runner file:
- `tester.php`
